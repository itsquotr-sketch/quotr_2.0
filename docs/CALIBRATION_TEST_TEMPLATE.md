### Quick reference — when to fail vs partial

| Decision | Typical signal |
|----------|----------------|
| **pass** | Scope, estimate, pricing, and quote align; minor polish only |
| **partial** | Correct with manual work areas, fact edits, or pricing tweaks |
| **fail** | Wrong scope path, unusable numbers, or missing capability |
